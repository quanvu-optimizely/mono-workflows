# Story Examples

This directory is for project-specific story examples.

Add one or two real completed stories here so future sessions have concrete
references for the correct format and level of detail.

## Recommended examples to add

- `STORY-001-example.md` — A data layer story (your first implemented story)
- `STORY-001-context-example.md` — The matching `context.md` after all tasks complete

## Why examples matter

The `story-creator` skill reads these files to calibrate output for your specific
stack, entity vocabulary, and path conventions. Generic examples are less useful
than real ones from your project.

## Format

Copy from a completed story in `.ai/stories/` — use a finished story where all
tasks have been executed and `context.md` is fully populated.
